About this course 

This course is designed to help you learn Japanese. It is a beginner's course and is designed to be easy to follow.
We will learn the basics of the language, we will learn to understand some basic Japanese
Reading, will also be part of the course however this is optional - it you are not interested in reading just skip lessons with the Icon
Lessons will have the following icons
- vocabulary
- listening
- speak
- reading
- grammar 
- culture
- writing
Feel free to skip elements you are less interested in
You can always come back to the lessons you have missed. 
The course is designed to work for a complete beginner - however you can go into more advanced modules if you feel that you can
